Voice-to-text that runs entirely on your Mac. Hold a hotkey, speak, your words land wherever your cursor is.

[**Download**](https://github.com/irangareddy/wishper-app/releases/latest/download/Wishper.dmg) · [Website](https://wishper.irangareddy.in)

## Features

- 100% on-device — no accounts, no network calls
- Works in Slack, Mail, VS Code, Notes, and anywhere else you type
- Filler-word cleanup via a local LLM
- Free and open source

## Requirements

macOS 15+ · Apple Silicon · ~2 GB free disk for models (downloaded on first run)

## Build from source

```bash
git clone https://github.com/irangareddy/wishper-app.git
open wishper-app/WishperApp.xcodeproj
```

Requires Xcode 16+.
